uint32_t* sofs20::soGetReferenceBlockPointer ( uint32_t  rbn)

Get a pointer to a reference block of the reference table.

Load from disk a reference block, if not done yet, and return a pointer to it.

A sofs20 exception is thrown if the dealer is not open.
